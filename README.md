# Common

Esse é um pacote escrito para servir de de base comum a todas as aplicações do Mola Corban.

## Estrutura

```
src/
tests/
vendor/
```


## Instalação

Via Composer

``` bash
$ composer require mola/common
```

## Testes

``` bash
$ composer test
```
